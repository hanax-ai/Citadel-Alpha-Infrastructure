## 📝 Template Instructions

**How to use this template:**

1. **Replace all placeholders** in square brackets with actual values
2. **Update the task execution status table** as tasks are completed
3. **Modify phase names** to match your specific project phases
4. **Add or remove tasks** as needed for your specific use case
5. **Update task status** using the following format:
   - ❌ Not Started
   - 🟡 In Progress
   - ✅ Complete
   - ⚠️ Blocked
   - 🔴 Failed

**Task Status Guidelines:**
- Each task should follow SMART+ST principles
- Include specific, measurable success criteria
- Provide clear validation procedures
- Document all dependencies accurately
- Estimate durations realistically

**Documentation Standards:**
- Keep objectives clear and specific
- Use consistent formatting throughout
- Update status regularly
- Document any deviations or issues
- Maintain version history
